## ¿Que es una base de datos?

una base de datos es un conjunto de informacion en un programa para la organizacion de este, osea toda nuestra informacion anotada y organizada en un programa para la facildad de su uso

## ¿por qué es en el ejemplo visto en clases. es mejor usar float a cambios de int en el ingreso del campo edad? 

por que float a diferencia de int permite utilizar numeros decimales, int solo nos permitiria usar numeros enteros, float esta diseñado especificamente para este tipo de situaciones en el que necesitamos un campo que incluya la edad

## ¿por que ingresar edad en un sistema de base de datos no es recomendable y cual seria una mejor opcion en este caso justifque?

no seria beneficiario para nuestro programa ya que al poner la edad, cada año se deberia actualizar esta informacion, seria recomendable colocar la fecha de nacimiento para asi calcular la edad con esta en ves de cambiar nuestro programa cada año
